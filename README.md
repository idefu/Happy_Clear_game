**快乐消消乐** 是一款基于前端现代化技术栈（React 19 + TypeScript + Vite 6 + Tailwind CSS 4 + Motion）开发的高品质、极简、高帧率的三消益智游戏（Match-3）。游戏以丰富的动画效果、连锁爆炸反应、精妙的关卡机制、本地高分星级评定、多国语言本地化（i18n）以及灵动音效为核心，提供了沉浸式的爆爽消除体验。

## 🌟 项目特性

**快乐消消乐** 的核心设计理念是**“极简、爆爽、极速反应、无缝出海”**。我们在传统三消游戏的基础之上，引入了一套极富策略性的炸弹与特殊道具合成机制，搭配炫酷的粒子发射器动画（Canvas Particle）和流畅的动效过渡，旨在为玩家提供卓越的触感与视觉反馈。

本游戏不仅是一个完整的、极具可玩性的消遣游戏，也是学习 React 19 新特性、复杂网格状态更新、矩阵重力填充算法、二维碰撞检测以及自定义 Canvas 高帧率渲染的绝富启发性范例。

### 1. 沉浸式大厅主干
* **多层暗色遮罩背景**：主页以高清 `pic/start.png` 作为底层充盈铺设，在中央辅以高斯模糊与深色微透渐变，将炫目的游戏标题与核心操作面板显露无遗。
* **缤纷多彩的糖果艺术字**：对大厅中央“快乐消消乐”主标题进行了亮彩、高饱和度的多重霓虹糖果渐变配色升级，释放出满满的童趣与元气。

### 2. 纸板箱 (Cardboard) 艺术风格关卡地图
* **精美质感**：关卡地图融入了独特的“黄褐色包装牛皮纸箱”拼贴风格，中央配以微透白封条质感，使关卡按钮犹如纸盒上排列整齐的印章。
* **高分星级反馈**：关卡下方支持最多 3 颗点亮星星。星星根据玩家历史最佳消算分动态点亮，具有极强的挑战欲和多重关卡回刷留存度。
* **锁死状态与锁形图标**：采用半透明厚重古铜底色辅以 Padlock（金属锁）挂机图标，清晰说明状态，并支持自动解锁下一关。

### 3. 多渠道赞助及爱心弹窗
* **完美比例赞助按钮**：在主页面、挑战成功页、挑战失败页等处优雅排布了双通道收款方式按钮（支付宝 `Alipay`、微信 `Wepay`）。

## 🎮 核心玩法与特殊道具合成

### 🧠 基本规则
通过鼠标点击拖拽或手指轻扫对应的元素：将其与**上下左右**相邻的元素交换。
* 只要交换后能在垂直或水平方向上连成 **3 个或以上相同类型元素**，即可引爆消除！

### 🚀 特殊棋子合成机制
在消除过程中，一次消除大于 3 个或构成不规则构型时，会合成本作最核心的**爆破配方**：
* **`BOMB 拆炸弹`**：如果触发了**平行消除**（例如两排独立的三消），那么在交点位置会合成为威力极悍的【炸弹药方 💣】。再次消除它时，将以其为核心撕裂出半径为 **2 格的超大范围物理范围爆炸**。
* **`0.png 黑洞 (HYPER_EXPLODER)`**：若当次爆破完成了**垂直交叉重叠消除**（例如 L 字形消、T 字形消或十字消），即会自动合成特制的 **0.png 黑洞棋子**。激活时会产生全屏漩涡引力尘埃，摧毁周遭所有阻碍。
* **`行/列光束 (Blasters)`**：进行 **4 个同色棋子直接连消**时生成（分别对应 `ROW_BLASTER` 与 `COL_BLASTER`）。引爆它时朝十字两侧发出夺目高能激光，瞬间扫清整条地块或整列格子。

### ❄️ 天才冰裂雷达定位仪 (Ripple Beacon)
* 关卡中某些特殊位置覆盖着坚厚的“双层 & 单层碎冰”。
* 为防止玩家视疲劳，我们在右侧指标卡上打造了极具创意的 **雷达定位器**：玩家随时在局内可以点击 **❄️ 霜层** (或者 `冰层/消除`) 目标按钮，整张网格内**残余未清除的碎冰格子**会瞬间闪烁粉红高亮，并向外辐射涟漪声波效果。定位体验直接拉满！

## 💡 核心功能实现原理

### 1. 递归多消与炸弹合成
当棋子滑动换回：
1. 游戏引擎通过双向扫描，在 `scanMatches` 阶段寻找同一轴线同色元素数量 $\ge 3$ 构成匹配团。
2. 结算消解时，由 `processMatchesAndGenerateSpecials` 探查是否存在交叉、构成 T/L 字形等复杂情况，在重影碰撞点上产生特殊配方棋子（如 `BOMB` 或 `HYPER_EXPLODER`）。
3. 消灭地块，对有消尽棋子的垂直下落由滑落双探照指针无阻力填充。

### 2. Canvas 飞溅粒子（60 FPS 性能保障）
为了规避由于过多的 DOM 碎块旋转、形变而造成的动画撕裂或手机卡顿，我们自主设计了一套高阶纯原生 **Canvas 粒子渲染器**：
* 每次发生引爆或普通消除时，立即向内部高精度粒子池塞入基于该单元格位置的几十个发散微粒。
* 特殊色彩：BOMB 炸弹喷涌出热带烈红火星，0.png 黑洞则激发迷离的彩虹渐变。
* 微秒更新通过 `requestAnimationFrame` 自主衰落、膨胀和消失，确保能轻松跑满极限帧率率高分屏，给玩家最丝滑流畅的爆爽触感。

## 🚀 安装与运行

### 1. 开包定位
确保当前终端路径处于包含 `package.json` 的项目根目录下。

### 2. 下载必要技术栈
运行 npm 安装指令以下载 React 19、Tailwind CSS v4 等开发依赖：
```bash
npm install
```

### 3. 一键在本地热加载体验
```bash
npm run dev
```
开发控制台输出：
```text
  vite v6.2.3  ready in 200 ms
  ➜  Local:   http://localhost:3000/
```
访问 http://localhost:3000/ 即可马上开启爆爽消除之旅！

### 4. 生产完美纯前端打包
```bash
npm run build
```
执行完毕后会在根目录生成静态包文件夹 `dist/`，该内容可直接上传任何服务器或用于离线游玩。

## ❓ 常见问题 (FAQ)

#### 🚩 Q1: 切换语言时，国旗有特定的显示排版吗？
* **A**: 是的。契合您的视觉审美要求，我们将 10 国国旗紧凑而高对称地编排成两排高斯模糊卡片。鼠标悬停时会反馈多语提示，点击即无感完成全局文字、提示与数值单位的本地化多国语言切换。

#### 💣 Q2: 滑动冰块目标定位，为什么会闪烁？
* **A**: 这是我们特意精雕细刻的 **Ripple Radar 霜层雷达寻迹系统**！玩家只需轻松触碰计分板上的碎冰目标标识，所有残存坚冰地块便会自发在网格中辐射红粉冷光波涟漪，帮助玩家在最后几步中快速找准清除位置。

#### 🎵 Q3: 为什么音频音量或爆炸声无法在移动端首次加载时复现？
* **A**: 移动端和电脑安全策略相同，需页面交互激活声卡。只需在大厅点击任意按钮、国旗或者在关卡随便滑一下，由 sound manager 驱动的原生多路混音环境就会立刻上线！
