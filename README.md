**快乐消消乐** 是一款基于前端现代化技术栈（React 19 + TypeScript + Vite 6 + Tailwind CSS 4 + Motion）开发的高品质、极简、高帧率的三消益智游戏（Match-3）。游戏以丰富的动画效果、连锁爆炸反应、关卡机制、本地高分星级评定以及音效为核心，提供了沉浸式的爆爽消除体验。

## 🌟 项目介绍

**快乐消消乐** 的核心设计理念是**“极简、爆爽、极速反应”**。我们在传统三消游戏的基础之上，引入了一套极富策略性的炸弹与特殊道具合成机制，搭配炫酷的粒子发射器动画（Canvas Particle）和流畅的动效过渡，旨在为玩家提供卓越的触感与视觉反馈。

本游戏不仅适合作为消遣的小游戏，也是学习 React 19 新特性、复杂网格状态更新、矩阵重力填充算法、二维碰撞检测以及自定义 Canvas 渲染的绝佳开源项目范例。

## 🎨 界面视觉全新改版与优化

为了给玩家提供更地道、更符合直觉、更精致的视觉感受，我们对游戏界面进行了全面的重构和汉化优化：

### 1. 对局界面中文化与符号精简
- 对局顶部的各项指标完成了精致的本地化重构：
  - 用 **`LV`** 代替旧版的 `STAGE`（如 `LV 1`）。
  - 用 **`目标`** 代替旧版的 `GOAL`。
  - 用 **`步数`** 代替旧版的 `Moves`。
  - 用 **`得分`** 代替旧版的 `pts`。

### 2. 通关成功与失败界面本地化
- 对局高分盘点以及结果弹窗彻底升级：
  - 用 **`LV`** 代替 `STAGE` 标头展示关卡。
  - 用 **`分数`** 代替 `Score` 汇报局内成绩。
  - 用 **`奖励`** 代替 `Bonus` 说明剩余步数转化的额外分值。
  - 用 **`总计`** 代替 `Total` 给玩家呈现最终的精算数值。

### 3. 全新沉浸式大厅背景与遮罩适配
- **背景图片重写支持**：游戏大厅采用了全新的 `pic/start.png` 进行充盈覆盖，配合多重模糊暗角遮罩，使标题与操作区对比度更鲜明。
- **卓越的多端布局适配**：精心编写了响应式 CSS 样式，自适应宽屏电脑浏览器、笔记本以及手机移动端屏幕。
- **渐进式交互逻辑**：隐藏了繁琐的列表，提供大厅核心双按钮服务：
  - 🌟 **开始对局**：一键进入玩家当前已解锁的最高进度关卡。
  - 📅 **关卡选择**：点击后平滑呈现独立的重构版关卡地图，界面更聚焦。

### 4. 复古纸箱板 (Cardboard Box) 关卡选择地图
- 采用 **纸质包装箱（Cardboard）** 风格底色（优雅的橙黄渐变结合中央透明封箱胶带设计），极具质感。
- **动态星级评定**：每一关下方支持最多 3 颗星星评价。星星会根据用户的本地最佳得分实时点亮，极具挑战与多关卡重玩价值。
- **右上角统计勋章**：精准显示玩家已赢取的星星总数（如 `33/75 ⭐`）。
- **锁与解锁状态**：未解锁关卡具有独立的挂锁标志（Padlock）与半透明古铜卡纸材质，而解锁关卡带有高对比调色盘与动画效果。
- **大红经典返回钮**：界面左下方保留了醒目的红色圆形经典返回手势动作按钮，极大还原经典掌机消消乐的童趣风貌。

## ✨ 核心功能

1. **多元化特殊道具合成**：
   * 🏆 **平行两列/两行三消升级【BOMB 炸弹】**：若当次消除中同时出现了两个平行的消去组，在对应位置将自动合成生成一张带有爆炸特效的炸弹卡片。
   * 🌟 **垂直相交三消升级【0.png 黑洞】**：若当次消除成功促成了至少两列/两行垂直相交相碰的图案（如 T 字、L 字或十字型），则会在交点位置自动生成一颗威力巨大的 **0.png (HYPER_EXPLODER)** 特殊棋子。
   * ⚡ **单行/单列四消升级**：生成对应的 `ROW_BLASTER` (行光束爆接) / `COL_BLASTER` (列光束爆接)。
2. **三维冰层消除与定位**：
   * 关卡包含带「双层/单层碎冰」的冰原地块。
   * **冰层定位仪 (Ripple Radar)**：玩家可在局内直接点击右侧看板的 ❄️ **碎冰层** 目标按钮，游戏棋盘内**残存未消除的冰层**会以高亮闪烁、雷达涟漪波纹（Ripple Beacon）和卡片轻微颤动等表现极速显现，方便快速通关。
3. **完美模拟物理重力**：
   * 消除完成后，上方棋子在重力驱动下按列自由滑落，并在上方自动补充全新生成的彩色棋子。
   * 具有递归的「连消（Combo）」链式触发评估，每多一次连消就会解锁更高的分数倍率与爽快的连续炸裂粒子。
4. **轻量、强韧的原生 Canvas 粒子反馈**：
   * 游戏不依赖臃肿的 WebGL 库，纯手写高效 Canvas Overlay 粒子喷发引擎。
   * 针对「BOMB (红色/橙色超快发散)」、「0.png (七彩霓虹彩虹漩涡)」与「普通消除 (匹配色飞散)」定制了独有的色彩衰减，维持 60 FPS 的流畅表现。

## 💡 功能实现原理

### 1. 关卡重力矩阵
在 `src/utils/gameLogic.ts` 中维护一个二维数组 `GridCell[][]`。核心算法依靠以下三步管线：
* **第一步：`scanMatches`**：按行、按列扫描匹配。检测连续相同值 $\ge 3$ 的项。
* **第二步：`processMatchesAndGenerateSpecials`**：评估是否有高阶匹配并消解格子。
* **第三步：`applyGravityAndSpawn`**：每列采用双指针探照：从底部开始往上填补，遇到空位则将上方的有效格下移，顶部缺口采用关卡设定的允许元素范围（`allowedLetters`）进行随机充能并平滑划入。

### 2. 精确地块高亮
为了实现最直观的碎冰定位：
* `ScoreBoard` 在用户点击未完成目标时向 `App.tsx` 抛出 `onIceGoalClick` 信号。
* `App` 全局开启 `isIceHighlighted` 阶段布尔开关，在所有处于 `cell.isLocked = true` 的 DOM 节点上动态添加高亮雷达涟漪波纹与微振动，极大降低了由于视疲劳而无法看清残余冰层的用户困扰。

## 🚀 安装与运行

完成以下步骤，即可在本地和服务器极速运行本游戏：

### 1. 下载或解包项目
保证当前处于项目的根目录下（包含 `package.json` 的文件夹）。

### 2. 开发依赖项安装
运行 npm 安装指令以下载 React 19、Tailwind CSS v4 等资源：
```bash
npm install
```

### 3. 本地启动开发服务器
```bash
npm run dev
```
启动成功后，将会在控制台输出：
```text
  vite v6.2.3  ready in 250 ms
  ➜  Local:   http://localhost:3000/
```
打开浏览器访问 [http://localhost:3000/](http://localhost:3000/) 即可马上体验！

### 4. 生产构建打包
当开发阶段结束后，运行如下指令执行编译打包：
```bash
npm run build
```
编译产物会无损生成于 `/dist` 文件夹内。你可以直接将 `/dist` 下的内容分发或托管在任何主流静态托管平台上。

## 🎮 使用说明与核心玩法

### 🧠 基本规则
- 🎯 **消除机制**：鼠标点击或手指触碰一个棋子，并将其向**上下左右**相邻的格子拖动。如果两颗棋子交换后能让同一条直线上（横或竖）拼凑出 **3 个或更多相同款式**的棋子，即可完成交换并消除！
- 💥 **特殊奖励机制**：
  - **行/列消除光波**：进行 4 棋子连消后生成。下一次消除它时会激活激光，横向/纵向扫清整条线。
  - **BOMB 炸弹**：触发平行消除时自动获取。引爆时产生半径为 2 格的巨型范围爆炸！
  - **0.png 黑洞（HYPER_EXPLODER）**：由不规则的多向交叉五消或十字垂直相夹组合形成。触发时发生全屏连环星璇风暴，炸毁所有的随机特殊方块并为全屏充能。
- 🧊 **关卡目标**：每一关有专门的目标：
  1. **积分（Score）**：在规定的步数（Moves）内打破历史高分。
  2. **除冰层（Ice Layer）**：在带有冰霜的格子上成功进行消除引发碎冰层瓦解。

## ❓ 常见问题 (FAQ)

#### 🔥 Q1: 为什么要重设计大厅封面与关卡地图？
* **A**: 为了让游戏看起来更完整。新大厅隐藏了长长的列表，使第一印象干净利落，完美兼容高宽分辨率。纸箱板地图则加入了高分评定、闪亮星星以及游戏进度的回溯挑战性。

#### 🚀 Q2: 游戏运行卡顿，或者是动画帧率不平稳？
* **A**: 请确保你的浏览器没有开启省电模式或多标签挂起限制。本游戏采用单层高效 Canvas Overlay 重绘粒子机制，日常渲染能轻易拉满 60-120 FPS 的显示上限。

#### 🎵 Q3: 为什么游戏启动没有声音？
* **A**: 设备通常要求用户先在页面上产生点击动作，声卡通道随之激活。在页面任何位置点击，音效即可恢复。
